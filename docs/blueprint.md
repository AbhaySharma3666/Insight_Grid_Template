# **App Name**: BI CanvasForge

## Core Features:

- Canvas & Background Customization: Adjust the overall canvas dimensions, background color, apply color gradients, or upload custom background images to the Power BI template.
- Flexible Grid Editor: Add, remove, and precisely customize rows and columns within the main canvas area, including adjustable sizes, spacing, borders, and shadows.
- Dedicated Side Panel Design: Integrate and design a customizable side panel (left or right) with its own internal grid system for navigation or specific data segments.
- AI Layout Suggestion Tool: Utilize an AI tool to suggest optimal grid and panel layout structures based on user-defined dashboard purpose (e.g., 'sales overview', 'executive summary').
- Interactive Design Preview: Provide a live, responsive preview of the Power BI background template as design changes are made.
- High-Resolution Image Export: Download the final Power BI background template as a high-resolution image file for easy integration into dashboards.

## Style Guidelines:

- Primary color: A sophisticated, muted blue (#2E62C9) to evoke data intelligence and clarity in a professional design context. (HSL: 220, 60%, 45%)
- Background color: An extremely light, desaturated grey-blue (#ECF0F7) to provide a clean, expansive workspace that is easy on the eyes. (HSL: 220, 20%, 95%)
- Accent color: A vibrant, clear cyan (#31C9D7) used sparingly for interactive elements, highlights, and call-to-action buttons to ensure high visibility and user engagement. (HSL: 190, 80%, 60%)
- Headline and body font: 'Inter' (sans-serif) for its modern, highly legible, and professional aesthetic, ensuring clarity across all text elements in the design tool.
- Clean, outlined, and minimalist icons to represent design functionalities, tools, and layout controls for intuitive navigation.
- An intuitive split-panel layout featuring a prominent, expansive design canvas on the right and a dedicated, organized control panel on the left, offering easy access to customization options.
- Subtle and fluid animations for interactions such as resizing grid elements, dragging and dropping components, and transitioning between customization panels to enhance user feedback and a sense of direct manipulation.
